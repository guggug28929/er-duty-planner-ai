# ER Duty Planner AI — Step 8

เวอร์ชันนี้รวมการแก้ไขต่อจาก Step 7

## ฟังก์ชันใหม่

1. Excel
   - เสาร์–อาทิตย์และวันหยุดราชการ: คอลัมน์วันที่พื้นแดงอ่อน
   - วันพุธที่ไม่ใช่วันหยุด: คอลัมน์วันที่พื้นเขียวอ่อน
   - วันธรรมดาอื่น: พื้นขาว
   - ใช้ชุดสีประจำตัวที่กระจายโทนชัดขึ้นและลดการผสมสีขาว
2. กำลังคนวันหยุด
   - เลือกได้ว่าจะใช้จำนวนคนต่างจากวันทำการหรือไม่
   - กำหนดเช้า/บ่าย/ดึกแยกได้ ทั้ง Resident และ Staff
3. Condition priority
   - OFF บังคับ สีแดงเข้ม: เรียน/สอบ/Vacation — ห้ามผ่อน
   - ขอ Off สีแดงอ่อน: ผ่อนได้เมื่อผู้ใช้อนุมัติ
   - เลี่ยง สีส้ม: Soft preference
   - ขออยู่ สีเขียว
4. Results
   - สรุป Hard/Soft violations แบบจัดกลุ่มภาษาไทย
   - ปุ่มแนวทางแก้และให้ Exact Solver จัดใหม่
   - ปุ่ม Refresh ตารางบนหน้าผลลัพธ์
   - AI ทำเครื่องหมายแนวทางที่แนะนำ
5. Repair options
   - จัดใหม่โดยไม่ผ่อนเงื่อนไข
   - ยอมผ่อน เลี่ยง → ขอ Off โดยไม่แตะ OFF บังคับ
   - ยอมให้ 8-เว้น-8 ต่อเนื่องเกิน 3 เวร
   - ลดขั้นต่ำเวรเช้าวันทำการเฉพาะวันที่ผู้ใช้เลือก
6. แก้ Exact Solver
   - แก้บั๊กที่ทำให้กฎ ADJACENT_SHIFTS ไม่ถูกบังคับใน Solver
   - OFF บังคับยังห้ามฝืนทุกกรณี
   - ทางผ่อนทั้งหมดถูกส่งให้ Validator ตรวจซ้ำ

## หลัง Deploy

1. เปิดเว็บและ Hard Refresh
2. ตารางเก่าที่บันทึกจาก Solver เวอร์ชันก่อนอาจยังแสดง ADJACENT_SHIFTS เพราะ Autosave เก็บตารางเดิมไว้
3. กด Refresh ตารางบนหน้าผลลัพธ์ เพื่อให้ Exact Solver เวอร์ชันใหม่สร้างตาราง
