## Disclaimer

ก่อนกด Exact Solver ผู้ใช้ต้องติ๊กยืนยันว่า:

- โปรแกรมเป็นเพียงเครื่องมือช่วยจัดเวร ไม่ใช่การอนุมัติเวร
- ผู้รับผิดชอบต้องตรวจสอบข้อบังคับโรงพยาบาล ชั่วโมงทำงาน เวลาพัก ความพร้อมบุคลากร วันหยุด และผล Validator
- ผู้พัฒนาไม่รับผิดชอบต่อความเสียหายจากการใช้ผลลัพธ์โดยไม่ตรวจสอบ
- ควรใส่ข้อมูลส่วนบุคคลเท่าที่จำเป็น
- การใช้ AI จะส่งข้อมูลตารางที่จำเป็นผ่าน Backend ไปยังบริการ AI ที่ตั้งค่าไว้

การยืนยันถูกบันทึกใน Browser เครื่องนั้น และสามารถยกเลิกการติ๊กได้
