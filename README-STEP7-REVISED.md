# ER Duty Planner AI — Step 7 Revised

เวอร์ชันนี้รวมฟังก์ชันทั้งหมดของ Step 7 เดิม พร้อม:

1. Autosave เมื่อแก้ข้อมูล จัดเวร ย้ายเวร เปลี่ยนหน้า หรือปิดหน้าเว็บ
2. Saved Versions ตั้งชื่อและย้อนกลับได้สูงสุด 8 ชุด
3. สำรองอัตโนมัติก่อนล้างข้อมูล ก่อนนำเข้า JSON และก่อนเปิดเวอร์ชันเก่า
4. ปุ่ม **ดาวน์โหลด Excel** ในหน้า Step 6 ผลลัพธ์

## รูปแบบ Excel ที่ส่งออก

Sheet ชื่อ `เวร` ประกอบด้วย:

- คอลัมน์ วันที่ / เวรเช้า / เวรบ่าย / เวรดึก / Condition
- แต่ละวันเว้นอย่างน้อย 5 แถว เพื่อให้อ่านรายชื่อได้สบายตา
- Resident ระดับสูงกว่าอยู่ด้านบน และ Chief แสดงด้วยสัญลักษณ์ ★
- คนหนึ่งคนใช้สีประจำตัวเดียวกันทุกช่อง และแต่ละคนคนละสี
- พื้นหลังเฉพาะคอลัมน์วันที่:
  - วันหยุดราชการ = แดงอ่อน
  - วันพุธ = เขียวอ่อน
  - วันอื่น = ขาว
  - ถ้าวันพุธตรงวันหยุดราชการ ใช้แดงอ่อน
- Condition แสดง OFF, ไม่ Prefer, ขออยู่, OFF weekend และ Basic science/Activity
- ด้านขวามีตารางสรุป เช้า บ่าย ดึก รวม วันหยุด และหมายเหตุ/รูปแบบการวน
